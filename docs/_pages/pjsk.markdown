# PJSK表情包生成

这是一个生成PJSK表情包的功能。

指令：`/pjsk 人物+编号 文本（支持多行）`

**指令不用加`/生成`！！指令是`/pjsk`！！**

- 例：`/pjsk 初音6 纳尼`
- 例：`/pjsk mzk14 气死我了`
